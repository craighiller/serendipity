Serendipity
======

Submission for HackBright Hack for Social Good

Time: 5/3/14 - 5/4/14 (2 days)

Designed to allow people to make wishes and fulfill them with the intent of charity donations fueling the system. Python Bottle.

App is currently running live at http://hackserendipity.appspot.com/
(can use username - Kevin, password - password to sign in or sign up)

Usage
---------------------
  - sign in/sign up
  - Make and fulfill wishes
  - Check how much money you've donated/raised!
